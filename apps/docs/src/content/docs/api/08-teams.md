## Overview

## Planned Endpoints

- `team.create` - Create new team
- `team.getById` - Get team details
- `team.getAll` - List user's teams
- `team.update` - Update team settings
- `team.delete` - Delete team
- `team.addMember` - Add team member
- `team.removeMember` - Remove team member
- `team.updateMemberRole` - Change member role
- `team.getMembers` - List team members
- `team.transferOwnership` - Transfer team ownership

**Full documentation will be added in Growth phase.**
