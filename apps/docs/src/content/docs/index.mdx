## API Overview

The Koko API is built with **tRPC** for end-to-end type safety, ensuring your frontend and backend stay in sync automatically.

### Key Features

- ‚úÖ **Type-Safe APIs** - Full TypeScript coverage from server to client
- ‚úÖ **Real-time Collaboration** - WebSocket support for live updates
- ‚úÖ **Video Processing** - Automated transcoding and streaming
- ‚úÖ **Secure by Default** - Built-in authentication and authorization
- ‚úÖ **RESTful Patterns** - Familiar query/mutation structure

### Technology Stack

| Layer | Technology |
|-------|-----------|
| **API Framework** | tRPC v11.5.0 |
| **Server Runtime** | Hono v4.8.2 |
| **Database** | SQLite/Turso with Drizzle ORM |
| **Authentication** | Better-Auth v1.4.0 |
| **Frontend** | React 19 + TanStack Router |
| **Type Safety** | TypeScript 5.8 (strict mode) |

## Getting Started with the API

### 1. Authentication

All API requests require authentication. Start by signing up and obtaining a session:

```typescript
import { trpc } from '@/lib/trpc';

// Sign up
const user = await trpc.auth.signUp.mutate({
  email: 'user@example.com',
  password: 'SecurePass123!',
  name: 'John Doe',
});

// Sign in (session cookie automatically set)
await trpc.auth.signIn.mutate({
  email: 'user@example.com',
  password: 'SecurePass123!',
});
```

### 2. Create a Project

```typescript
const project = await trpc.project.create.mutate({
  name: 'My Video Project',
  description: 'Product demo videos',
});
```

### 3. Upload a Video

```typescript
const video = await trpc.video.createUpload.mutate({
  projectId: project.id,
  fileName: 'demo.mp4',
  title: 'Product Demo V1',
});

// Upload file using the provided upload URL
// (See Videos API documentation for full upload flow)
```

### 4. Add Comments

```typescript
const comment = await trpc.comment.create.mutate({
  videoId: video.id,
  text: 'Great shot! Consider adjusting the lighting.',
  timecode: 45.2, // 45.2 seconds into the video
});
```

### Local Development

